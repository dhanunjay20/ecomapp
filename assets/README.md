# Assets Directory

This directory should contain the following image assets for your app:

## Required Assets

### App Icon
- **icon.png** (1024x1024 px) - Main app icon

### Splash Screen
- **splash.png** (1242x2436 px) - App splash screen shown during loading

### Android
- **adaptive-icon.png** (1024x1024 px) - Android adaptive icon foreground

### Web
- **favicon.png** (48x48 px or 192x192 px) - Web favicon

### Notifications (Optional)
- **notification-icon.png** (96x96 px) - Custom notification icon for Android

## How to Add Assets

1. Create or design your app icons using tools like:
   - [Figma](https://figma.com)
   - [Canva](https://canva.com)
   - [Adobe Illustrator](https://adobe.com/illustrator)

2. Use online generators to create required sizes:
   - [App Icon Generator](https://www.appicon.co/)
   - [Expo Asset Generator](https://romannurik.github.io/AndroidAssetStudio/)

3. Place the files in this directory with the exact names listed above

## Temporary Solution

For development, you can:
1. Download placeholder icons from [Icon Archive](https://iconarchive.com/)
2. Use Expo's default assets by removing the asset paths from app.json
3. Generate simple colored squares using online tools
